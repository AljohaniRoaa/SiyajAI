# GovernAI: PDPL & NDMO Compliance Checker

**GovernAI** is an AI-powered compliance analysis tool designed to help Saudi organizations align their data assets with the **Personal Data Protection Law (PDPL)** and **NDMO Data Management & Governance Regulations**.

## 🚀 Key Features
- **Sensitive Data Auto-Detection:** Automatically flags PII (National ID, Phone Numbers, Emails) using pattern matching and heuristics.
- **PDPL Assessment Engine:** Evaluates dataset structures against regulatory constraints.
- **NDMO Scorecard:** Generates compliance scores and actionable gap analyses.
- **Exportable Audits:** Download PDF/Excel compliance sheets.

## 🛠️ Tech Stack
- **Frontend/UI:** Streamlit
- **Data Engine:** Pandas, NumPy, OpenPyXL
- **Language:** Python
